# Transcribator

Локальный сервис и desktop-приложение для транскрибации видео и аудио в текст с таймкодами, структурированным разбором встречи и экспортом результата в удобные форматы.

## Что умеет

- принимает аудио и видеофайлы через браузер или desktop-оболочку;
- поддерживает drag-and-drop загрузку;
- извлекает аудио из видео и подготавливает его через `ffmpeg`;
- делает транскрибацию через Deepgram;
- по желанию строит summary встречи через OpenAI;
- выделяет TL;DR, цель созвона, краткий пересказ, решения, договоренности, открытые вопросы и задачи;
- показывает таймкоды по главам и полный транскрипт;
- позволяет скачать результат в `.md`, `.txt`, `.srt` и `.json`;
- поддерживает две темы интерфейса: `STILT` и `NEON`;
- дает локальное окно настроек API-ключей и моделей в desktop-версии.

## Технологии

- Node.js
- Express
- Multer
- Deepgram API
- OpenAI API
- ffmpeg-static
- ffprobe-static
- Electron

## Быстрый старт

1. Установи зависимости:

```powershell
npm.cmd install
```

2. Создай `.env` на основе шаблона:

```powershell
Copy-Item .env.example .env
```

3. Заполни ключи и настройки:

```env
DEEPGRAM_API_KEY=your_deepgram_api_key_here
OPENAI_API_KEY=your_openai_api_key_here
PORT=3100
DEEPGRAM_MODEL=nova-3
DEEPGRAM_LANGUAGE=ru
SUMMARY_MODEL=gpt-4o-mini
MAX_UPLOAD_MB=512
AUDIO_BITRATE=48k
AUDIO_SAMPLE_RATE=16000
```

4. Запусти приложение:

```powershell
npm.cmd start
```

5. Открой [http://localhost:3100](http://localhost:3100)

Или просто запусти [start-app.bat](c:\Users\Askay\Desktop\AI\transcribator\start-app.bat): он поднимет сервер и откроет приложение автоматически.

## Desktop-версия

Запуск в Electron:

```powershell
npm.cmd run desktop
```

Сборка portable `.exe`:

```powershell
npm.cmd run dist:win
```

Готовый portable-файл появляется в папке `dist`. Его можно передать коллеге как локальное desktop-приложение без отдельной установки Node.js и зависимостей проекта.

## Экспорт результата

После обработки файла можно скачать отчет в одном из форматов:

- `Markdown (.md)` для структурированного отчета;
- `Text (.txt)` для простого текстового экспорта;
- `Subtitles (.srt)` для таймкодов и глав;
- `JSON (.json)` для интеграций и дальнейшей автоматической обработки.

## Основные переменные окружения

- `DEEPGRAM_API_KEY` — ключ для транскрибации;
- `OPENAI_API_KEY` — ключ для summary;
- `PORT` — локальный порт приложения, по умолчанию `3100`;
- `DEEPGRAM_MODEL` — модель транскрибации, по умолчанию `nova-3`;
- `DEEPGRAM_LANGUAGE` — язык распознавания, по умолчанию `ru`;
- `SUMMARY_MODEL` — модель для summary, по умолчанию `gpt-4o-mini`;
- `MAX_UPLOAD_MB` — лимит входного файла на сервере;
- `AUDIO_BITRATE` — битрейт подготовленного аудио;
- `AUDIO_SAMPLE_RATE` — sample rate подготовленного аудио.

## Ограничения

- для транскрибации нужен доступ в интернет;
- для summary нужен отдельный доступ к OpenAI API;
- если summary отключен чекбоксом, приложение возвращает только транскрипт и таймкоды;
- качество результата зависит от исходной записи, шума и разборчивости речи.
